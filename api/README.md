#### New update URLs for chrlauncher

1. Open your chrlauncher folder, open `chrlauncher.ini` file with notepad

2. Delete `#` on `ChromiumUpdateUrl`

3. Then replace the update URLs (something like this: `https://chromium.woolyss.com/api/v3/?os=windows&bit=%d&type=%s&out=string`) with URLs below based on your installed variants
